An introduction to CSS

Customising a HTML/ document page requires the use of the programing language CSS (Cascading style sheets) so we can change how the document is displayed to the user.

To implement your CSS file in a certain html file, we use the link element which has the self closing tags (as in we do not need to use /link to close the element), within the head element.
We follow with the attribute rel(relationship)="stylesheet" and then use the href attribute to define the location.

For example for this document I have used 'link rel="stylesheet" href="cssNotes.css"'

Also we can use the 'style' tag within the element to change it individually within the HTML file.

As I will discover forming this note webpage, there is a lot that you can do to change the style within different elemnts of the webpage.

Using CSS

Once you want to start customising your html page the rules are pretty simple.
You first select what element, class or ID you want to use. Then open and close with { } putting all the context within the curly brackets.

When targetting the different catergories there is a few rules to follow as seen bellow;

  • Targetting general elements such as P or h1, you define them as they are before the brackets. 'p {}'
  • A class needs a full stop '.' before the class name. E.g. '.class{}'
  • An ID shares the same princple with a class except it needs a '#' before it. Like so, '#id {}'

Color

We can change the color in certain ways, such as;

  • background-color: - The background color of divs for example
  • color: - Changes the text color

You can select a color by..

  1. Typing the name of it, e.g. 'white' and 'black'
  2. Using the hex color code, e.g. '#ffffff' (white) and '#000000' (black)
  3. Typing in the RGB code, e.g 'rgb(255, 255, 255) - whtie
    • Each number representing a certian value of red, green and blue
